# Security Policy

## Reporting a Vulnerability

We value the security of the Orion-OS framework and welcome reports from the security community. If you discover a potential vulnerability or architectural weakness, please disclose it to us privately so we can address it before it is exploited.

Please report all security-related findings to:

**Contact:** theindigoxcollectionx@gmail.com

We commit to acknowledging your report and keeping you updated on the status of the remediation. We appreciate your efforts in helping us maintain a secure architecture.
